# Road_Safety_Model_Submission
This repository collects all the deliverables for TeamGroot's submission to ADB's AI for Safer Roads Innovation Challenge.

## Setup and Run Instructions
We assume that viewers/judges have access to the provided datasets and thus in accordance with the Fair Data User Agreement, we haven't hosted them here.

### Viewing Code and Outputs 
By viewing the .ipynb files on this page (or by clicking the link to view them on Google Colab), you can see the code blocks and their respective outputs.

### Running .ipynb files
The Analytical Model, Speed Safety Score and (the code for) the Geospatial Visualisation were written in Google Colab and that is same platform we recommend for running the files. All notebooks have certain prerequiste files that need to be uploaded at runtime:
1. [Analytical Model](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/AnalyticalModelTeamGroot.ipynb):
ADB_Innovation_Maharashtra.geojson, ADB_Innovation_Thailand.geojson, [maharashtra_sss_scored.geojson](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/maharashtra_sss_scored.geojson), [maharashtra_vru_exposure_output(2).geojson](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/maharashtra_vru_exposure_output(2).geojson)
2. [VRU Exposure Index Calculation](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/VRUExposureIndexCalculation.ipynb): ADB_Innovation_Maharashtra.geojson
3. [Speed Safety Score](): ADB_Innovation_Maharashtra
4. [Geospatial Visualisation](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/geomap_vru_ss.ipynb): [maharashtra_sss_scored.geojson](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/maharashtra_sss_scored.geojson), [maharashtra_vru_exposure_output(2).geojson](https://github.com/samrudhisinghk/Road_Safety_Model_Submission/blob/main/maharashtra_vru_exposure_output(2).geojson)
